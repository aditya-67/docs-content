### Update Inventory API

Updates the inventory quantity for an existing catalogue item.

**Sample Request**

<table>
  <tr>
    <th>Method</th>
    <td>
      <code>PATCH</code>
    </td>
  </tr>
  <tr>
    <th>Path</th>
    <td>
      <code>
        /api/storefronts/{"<storefront_id>"}/catalogue/{"<catalogue_item_id>"}
        /inventory
      </code>
    </td>
  </tr>
</table>
<br />

**Sample Request Body**

<CodeBlockWithCopy>
  {`
{ 
    "quantity": 3000 
}
    `}
</CodeBlockWithCopy>

**Sample Response**

<CodeBlockWithCopy>{`202-EMPTY RESPONSE`}</CodeBlockWithCopy>
